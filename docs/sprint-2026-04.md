# Sprint Report — April 2026

## Summary

Major feature sprint covering live player stats, recurring availability, scrim/community registration portals, and full Discord bot integration. The site went from a static roster display to a connected platform with live data, public-facing forms, and automated notifications.

## Features Delivered

### Live Marvel Rivals Player Stats
- Integrated Marvel Rivals API for live ranked stats on the public team page
- Hero portrait images from Blizzard CDN (OW2, 256×256 square) and marvelrivalsapi.com headshot icons (MR, 344×344 square)
- Manual stats override system for admins (player_stats_override table)
- Filters out unrecognized heroes ("Unknown" from API) gracefully
- Static hero icon mapping with API thumbnail fallback for future-proofing

### Recurring Availability System
- **Weekly templates:** Players set a 7-day × 3-block (Morning/Afternoon/Evening) recurring schedule once
- **One-off overrides:** Date-specific exceptions that auto-prefill from the template
- **Resolution logic:** Override > Template > Unset — computed dynamically, no pre-generation
- **"My Schedule" tab** added to /team-hub/availability alongside the existing weekly grid
- Role-gated: players edit own, OWNER/ORG_ADMIN can edit on behalf of others

### Scrim Application Portal
- **Public form** at `/scrims/apply` — no login required
- Auto-sets format: Marvel Rivals → 6v6, OW2 → 5v5
- Duplicate guard: same captain + game within 7 days
- **Applications tab** in `/team-hub/scrims` for Manager+ with Accept/Decline actions
- Discord bot notification on submission with team availability overlay for requested days

### Community Team Registration
- **Public form** at `/en/community/join` (bilingual EN/ES via i18n)
- 3-section form: Team Info → Players (captain + up to 9) → Final Details
- Dynamic player rows with add/remove, pill toggles for games/platforms/goals
- Duplicate guard: same captain + overlapping games within 30 days
- **Admin view** at `/team-hub/community` — approve/decline, expandable player details, internal notes
- Discord bot notification on submission and approval
- "Register Your Team" CTA on homepage (after "More Than a Team" section)

### Discord Bot Integration
- Bot connected to 4 dedicated channels:
  - **Team channel** — availability completion (day-by-day breakdown + best days), schedule changes, session reminders
  - **Scrims channel** — new scrim applications with team availability overlay
  - **Community channel** — new team registrations + approval welcome messages
- Vercel Cron set up (every 5 min) for automated session reminders (1hr + 15min)
